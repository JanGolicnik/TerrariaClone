# TerrariaClone
Project made for fun using OpenGL and a few libraries. There's crafting, world gen, lighting etc. No entities tho.
You can add custom blocks/items/recipes by tinkering with the createBlocks() function in main.cpp and recompiling the app. 

A lot of the functions might be implemented a bit oddly, I was manily working on this while at sea without an internet connection so I made most non OpenGL things myself, and they may be a tiny bit..bad...ahem..world gen..
